# Todo_x

一个简约、优雅的 Flutter 记事应用，专为 Android 平台设计。

## 项目简介

Todo_x 是一个轻量级的本地记事应用，帮助你快速记录和管理日常事项。应用采用紧凑的 UI 设计，支持深色/浅色主题切换，让记事变得简单高效。

## 主要功能

### 📝 记事管理
- **快速添加**：简洁的输入界面，支持标题和详情记录
- **智能分组**：按日期自动分组，最新记事优先显示
- **置顶功能**：重要事项可置顶，独立分组展示
- **编辑修改**：专用编辑页面，大屏幕舒适编辑

### 🗂️ 组织功能
- **归档**：完成的事项可归档保存，保持主界面整洁
- **回收站**：删除的记事进入回收站，支持还原或彻底删除
- **批量清理**：一键清空回收站所有内容

### 🎨 界面特色
- **彩色标识**：每条记事用彩色圆点标识，视觉区分清晰
- **紧凑布局**：小字体设计，信息密度高，适合快速浏览
- **滑动操作**：左滑删除/归档，右滑编辑/置顶，操作流畅
- **主题切换**：支持浅色/深色模式，适应不同使用场景

### 💾 数据管理
- **本地存储**：所有数据存储在本地，隐私安全
- **数据导出**：支持导出为 JSON 格式，方便备份
- **数据清除**：可一键清除所有数据

## 技术栈

- **框架**：Flutter
- **状态管理**：Provider
- **本地存储**：SQLite (sqflite)
- **UI 组件**：flutter_slidable (滑动操作)

## 使用说明

1. **添加记事**：在"记事"页面输入标题和详情，点击右下角 + 按钮保存
2. **管理记事**：在"总览"页面查看所有记事
   - 左滑：删除或归档
   - 右滑：编辑或置顶
3. **查看归档/回收站**：点击总览页右上角图标进入
4. **切换主题**：在"设置"页面切换深色/浅色模式
5. **导出数据**：在"设置"页面导出所有记事为 JSON 文件
